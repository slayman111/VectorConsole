# VectorConsole

Приложение, позволяющее работать с векторами через консольный интерфейс

## Начало работы

1. Скачать NetFramework 4.8
2. Запустить VectorConsole.exe

### Необходимые условия

Что нужно для установки программного обеспечения и как его установить

### Установка

1. Скачать репозиторий
2. Открыть папку bin
3. Открыть файл VectorConsole.exe
